# ➕ Floating Social Menu UI

Um projeto moderno de **Menu Flutuante de Redes Sociais**, desenvolvido com foco em interatividade, animações e experiência visual utilizando **HTML5**, **CSS3** e **JavaScript**.

Ideal para websites, landing pages, portfólios e aplicações web que desejam integrar atalhos rápidos para redes sociais de forma elegante e dinâmica.

---

## 📸 Preview do Projeto

Interface contendo:

- Botão flutuante expansível
- Menu vertical animado
- Ícones de redes sociais
- Efeito de abertura e fechamento
- Animações suaves
- Layout moderno e minimalista

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome**

---

## 🎨 Características do Design

- Interface moderna e interativa
- Botão flutuante animado

- Animações fluidas
- Estrutura limpa e responsiva

---

## 🚀 Objetivo do Projeto

Praticar o desenvolvimento de componentes interativos para interfaces web, explorando menus expansíveis, animações CSS e manipulação dinâmica com JavaScript.

---

## 📚 Funcionalidades

- Abrir e fechar menu social
- Navegação rápida para redes sociais
- Animações suaves de transição
- Botão flutuante interativo
- Layout responsivo
- Estrutura reutilizável para projetos web
